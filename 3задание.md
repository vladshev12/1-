1.реализация

```python
    def bin_poisk(n,a):
    n.sort()
    l=0
    r=len(n)-1
    while r>=l:
        mid=(r+l)//2
        if a>n[mid]:
            l=mid+1
        elif a<n[mid]:
            r=mid-1
        elif a==n[mid]:
            return True
    return False
    
```
2.генерация массива
```python
    def generate_array(n):
    arr = []
    for i in range(n):
        arr.append(random.randint(0, 10000))
    return arr
```
3.измерение времени
```python
    def measure_time(func, data,a):
    start = time.perf_counter()
    func(data,a)
    end = time.perf_counter()
    return end - start
```
